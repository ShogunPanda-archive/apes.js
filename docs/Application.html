Application

staticApplication.execute(configurationRoot, main, root, prepare, cleanup){Promise.<(void|Error)>}

application.js, line 28

Creates and executes a new application. It's a short hand for Application.create followed by Application.run.

Name Type Default Description
configurationRoot string

The root key in package.json holding application configuration.
main function

The main application loop. It will receive a application object as the only argument.
root string null optional

The root directory of the application.
prepare function null optional

A optional function to run right before the main loop. It will receive a application object as the only argument.
cleanup function null optional

A optional function to run right after the main loop. It will receive a application object as the only argument.
Returns:
Type Description
Promise.<(void|Error)> Nothing in case of success, the error otherwise.
