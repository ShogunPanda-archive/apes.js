Logger

A module to perform rich logging, based on Winston.

Methods

staticLogger.create(application, target, useConsole, root){Promise.<object>|Error}

logger.js, line 28

Creates a new logger.

Name Type Default Description
application object

A application object (created via Application.create) the new logger will belong to.
target string

The target log file. It will be stored in the log folder with the current environment appended.
useConsole boolean false optional

Whether or not also log on console.
root string null optional

The root folder for the log file. Usually it's the log in the current working directory.
Returns:
Type Description
Promise.<object> | Error A new logger object in case of success, the error otherwise.
