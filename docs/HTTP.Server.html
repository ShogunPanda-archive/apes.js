addCORSHandling(origin, headers, methods, maxAge)

Adds CORS handling to the Express server.

Parameters:
Name Type Description
origin string

A value for the Access-Control-Allow-Origin header. If not provided, will be inferred by requests' Origin header.
headers string

A value for Access-Control-Allow-Headers header. If not provided, * will be used.
methods string

A value for Access-Control-Allow-Methods header. If not provided, only GET and POST methods will be allowed.
maxAge string

A value for Access-Control-Max-Age header. If not provided, one year will be used.
